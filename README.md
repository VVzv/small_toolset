# small_toolset
small toolset to use
#### hex_change.py:
用于十六进制与字符之间的相互转换,使用方法：
```
字母或者字符转16进制：
python3 hex_change.py abc
python3 hex_change.py " "
16进制转字母或者字符：
python3 hex_change.py 0x35
python3 hex_change.py 0x2021
```
#### hex_change_up.py：
hex编码强化版本，支持0x、\x的编码解码，支持\r、\n等特殊转换，使用方法通hex_change.py。
